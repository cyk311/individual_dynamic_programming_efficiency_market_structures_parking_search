# 不同市场结构下个体动态规划最优解的整体效率——以车位搜寻为例

### 本项目文件分为三部分：
1. 论文一篇《不同市场结构下个体动态规划最优解的整体效率——以车位搜寻为例》（PDF文件）；
2. Latex文件夹：包含上述PDF文档的编译文件（TEX格式文件），以及文档中的图表文件（EPS文件）；
3. Matlab文件夹：包含上述EPS图像文件的绘图代码以及数学建模代码文件（M文件）。
